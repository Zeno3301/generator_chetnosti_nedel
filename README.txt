Academic calendar
Генератор учебного календаря для университета с четностью недель.

Особенности
Правильная нумерация недель (первая неделя содержит 1 сентября)
Поддержка годов, где 1 сентября - воскресенье
Экспорт в CSV
Консольный интерфейс с параметрами

## Тестирование

1. Скачайте проект: `git clone https://github.com/Zeno3301/Academic-calendar.git`
2. Запустите программу: `python run_gui.py`
3. Используйте [тестовые сценарии](test_scenarios.md)

### Краткая инструкция для тестировщиков:
- Главная кнопка: "СГЕНЕРИРОВАТЬ КАЛЕНДАРЬ" - создает календарь
- Кнопка "ИНФОРМАЦИЯ О ГОДЕ" - показывает детали о выбранном годе
- "Экспорт в CSV" - сохраняет данные в файл
- "Очистить таблицу" - удаляет результаты

### Особые случаи для проверки:
